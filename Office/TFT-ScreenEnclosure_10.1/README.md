I was able to take over the display from a defective tablet. It is an 'Innolux 10.1" TFT Display EJ101IA-01G'. For this I got a replacement controller 'TFT-Controller PCB-800815'.
Since my 3d printer is too small, I had to print some elements in 2 parts. I then glued the parts together, using 2 pins each with D=3mm as guides.
For illustration I have attached the assembly as a step file.
